## Getting Started

Clone the repository:

```bash
git clone https://github.com/vivekkmrpaswan/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
